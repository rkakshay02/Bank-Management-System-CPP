# 🏦 Bank Management System (C++)

A menu-driven Bank Management System developed using **Object-Oriented Programming (OOP)** concepts in C++.

## 🚀 Features

- Create Bank Account
- Display All Accounts
- Search Account
- Deposit Money
- Withdraw Money
- Delete Account
- Menu-driven Console Interface

## 🛠 Technologies Used

- C++
- Object-Oriented Programming
- STL Vector
- Enum
- Constructors
- Encapsulation

## 📚 OOP Concepts Covered

- Classes & Objects
- Constructors
- Encapsulation
- Enum
- Member Functions
- Vector of Objects

## ▶️ How to Run

Compile

```bash
g++ main.cpp -o bank
```

Run

```bash
./bank
```

Windows

```bash
bank.exe
```
